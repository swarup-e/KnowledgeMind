# KnowledgeMind - Python dependencies
# ------------------------------------
# Install:  pip install -r requirements.txt
# Then:     python -m spacy download en_core_web_sm   (see spaCy note below)
#
# Constraints (see CLAUDE.md):
#   - Must install cleanly on Windows 11 x64 via pip.
#   - torch IS allowed: sentence-transformers requires it for embeddings.
#     tensorflow / torchvision / torchaudio are still excluded (not needed).
#   - Prefer stdlib over third-party for simple tasks (urllib, sqlite3 are NOT listed here).
#
# Versions are intentionally left unpinned for this scaffold. Pin them
# (pip freeze) once a known-good combination has been validated end to end.
